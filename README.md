## スポーツのリーグ戦などの出欠管理のアプリ

### 開発手法
- バージョン管理 Git/Github
- DB Supabase
- サーバー Versel
- Nuxt.js

### 必要な機能
- 監督、選手がユーザーとして登録できる。
- 監督がチームを作成できる。
- 選手はチームへの加入依頼を申請できる。
- 監督は選手の加入依頼を承認してチームに加入させる。
- 監督が試合のスケジュールを作成、管理できる。出欠確認。
- 選手は試合の出欠確認に参加可否を回答する。
- 監督は参加可能な選手の中でポジションを決定する。

### 画面一覧
#### 共通
- トップ画面
- ログイン
- 新規登録
- パスワード再発行
- チーム選択
- チームトップ画面
- プロフィール（プロフィール編集）
- チーム情報
- 試合スケジュール一覧
- 試合詳細情報

#### 監督
- 新規チーム作成
- 試合新規作成
- チーム情報（チーム情報設定）

#### 選手
- 新規加入チーム申請
